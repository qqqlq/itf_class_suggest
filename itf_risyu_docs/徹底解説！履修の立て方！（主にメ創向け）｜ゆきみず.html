1, 履修登録ってそもそも何？基本用語

履修要覧を読みましょう！！以上



・・・で終わる訳にはいきません。
履修要覧とは、簡単に言えば大学の履修のすべてが詰まった文書です。これを見れば全てのことが分かります。
ここからの話は概ねこれに沿いつつ、カジュアルな表現に置き換えて進めるので、厳密に知りたい点などは履修要覧を参照してください。

高校までの授業とは異なり、大学では自分が受けたい授業を選んで、オリジナルの時間割を作っていきます。だから授業のない時間帯も存在し、空きコマや全休（丸一日授業がないこと）なんて言葉が存在するわけですね。この授業を選び、登録する作業が履修登録です。

授業にはそれぞれ単位というものが定められています。単位とは、簡単に言えばその授業を修めた証であり、授業での成績が合格ラインに達すれば単位取得、成績が悪ければ取得できません。得単、落単などの用語はこのことです。落単は、その授業を履修していないのと同等に見なされてしまいます（この後の話で登場する「履修する」という言葉は、基本得単することが前提です）。取得できる単位数も授業によって異なり、1単位の授業や2単位の授業のように様々です。通年の授業だと3単位のものもあります。


そして、授業を選べるからと言って完全に自由なわけではありません。授業には主に以下の種類があります。

・専門科目：その学類の専門分野についての科目で、主に3、4年向けのもの。
・専門基礎科目：その学類の専門分野についての科目で、主に1、2年向けのもの。
・基礎科目-共通科目：英語、国語、体育など、いわゆる教養系。
・基礎科目-関連科目：他学類や特設自由科目など（つまりその他）

授業にはそれぞれには学類ごとに以下のような分類があります。
・必修科目 : 卒業までに絶対に履修しなければいけない科目
・選択科目（選択必修） : 授業群の中から、一定数を選んで履修しなければならない科目。

（自由科目 : 選択科目より広範囲から選ぶ。自由科目（特設）とは別物。卒業要件を私が見る限り有名無実化しているため、今回は気にしなくてよい。学類によっては影響あり。）

さらに複雑なのが、必修や選択必修を最低数だけ履修すれば良いというものでもないのです。
最低数を履修するのはある意味前提条件のようなもので、これを満たしてかつ卒業までに124単位を取得することで、晴れて卒業できるわけです。つまり、必修をすべてとり、選択の最低単位数を満たしつつ、合計124単位の取得が必要です。

大変だ！さらに留年の条件などもありますが・・・後述。

あと知っておくべき用語として、シラバスと開設授業科目一覧があります。

シラバスとは、授業の内容や時間、成績評価方法、単位数などがまとめられたものです。全ての授業にはシラバスが存在し、それを見ながら授業を選んでいきます。シラバスはkdbで科目検索をすることで見れます。
開設授業科目一覧は、その名の通り開設されている授業の概要一覧です。これを見ながら授業を探したりします。

ほかにも標準履修年次という言葉もあります。これは授業ごとに設定されていて、その授業が何年生向けかという指標になります。基本的に自分の学年に合った標準履修年次の授業を取りましょう。無視しても構いませんが、一部はその年次以外の履修を制限するものもあります。シラバスを見ましょう。

2, 筑波大の履修の特徴

筑波大学は2学期6モジュール制と呼ばれる年間スケジュールをとっています。履修登録はこのモジュールごとに行います。1モジュールあたり、基本5週間となります。

モジュールごとと言いましたが、履修登録自体はそのモジュールが来るまでならいつでも出来るので、年度初めに1年分全て登録するのが一般的です。

そして、開講される授業はモジュールごとに大きく異なります。
例えば、1年生全員必修の授業として、English Presentation Skills 1という授業があります。これは、春A、B、Cに渡って授業が行われます。一方、線形代数Aという授業は、春B、Cに渡って授業が行われます。さらに、情報メディア入門という授業は、春Aのみで授業が終わります。

このように、授業によってどのモジュールで開講されているかが異なっています。長いものだと通年のものもあります。

これらも考慮しつつ、時間割を組む必要があります。当然時間が被る箇所に授業は入れられませんから。

3, ではどんな授業を取れば良いの？

皆さんが気になるであろう、どの授業を取ればよいのか、そしてどうすれば卒業できるのかという問題。

その答えを記しているものが履修要覧です。正確に言うならば履修要覧の学群等履修細則の自分の学類のページです。
例えばメディア創成は以下のようになります（2024年時点、情報は古いものです）。

科目名や種類などがたくさん並んでいますね。科目の分類についてざっくり見ていきましょう。

これが必修等と対応させたバージョンです。赤が必修、青が選択です。

赤枠の必修から見ていきましょう。
左2つの赤枠は卒業研究、微分積分Aなど授業名が書かれていると思いますが、これらの授業は卒業までに絶対に履修しなくてはいけません。絶対に履修しましょう。

左から3つ目の共通科目の赤枠を見ると、このように書かれています。

総合科目（ファーストイヤーセミナー、学問への誘い）というのは、まさに"ファーストイヤーセミナー"と"学問への誘い"という2つの授業を指します。この2科目を履修してくださいという意味ですね。これらは1年生で取る授業です。

次に情報。これはここには詳しく書かれていませんが、とる授業が決まっています。それは、"情報リテラシー（講義）"、"情報リテラシー（演習）"、"データサイエンス"の3つです。これらを履修しましょう。これらも全て基本1年生で取る授業です。

続いて体育。これもここには詳しく書いていないのですが、基礎体育(春、秋)、応用体育(春、秋)の4科目を履修しなくてはなりません（学類によっては発展体育も）。こちらはオリエンテーション等で詳しく説明があるので、それを聞いてください。

最後に外国語（英語）。これは"English Reading Skills1,2"、"English Presentation Skills1,2"の4科目を指します。春が1、秋が2です。こちらは入学後に受験するTOEICによってクラス分けがされ、それをもとに履修登録を行います。こちらも、その説明を参照してください。これらも全て1年生で取る授業です。

なお、"English Presentation Skills1,2"は、英検準1級（学類によっては1級）などの英語資格を持っていると、授業を受けなくても単位を取得できます（単位認定）。認定方法は割愛。

この必修の共通科目については、履修要覧の"3 基礎科目（共通科目等）の履修方法"に詳細が書いていますので、ぜひそこを読んでみてください。

続いて選択必修。青枠は4つありますが、それぞれ表記が少々異なります。

左2つは科目欄に、"GC5, GA4"や"GC2, GA1"と書かれています。その横に単位数が"20~35", "32~47"と書かれています。

まず科目欄の記号。これは科目番号を示しています。科目番号は開設授業科目一覧、およびシラバスに掲載されています。

例として、開設授業科目一覧から情報メディア入門を持ってきました。

左端にあるのが科目番号です。GA1から始まっていますよね。科目欄にあった"GA1"とはこのことで、"GC2, GA1"とは、"GA1から始まる授業、またはGC2から始まる授業"という意味なのです。"GC5, GA4"などについても同じ考え方で大丈夫です。

そして、単位数。"32~47"というのは"最低32単位分を履修してください。47単位まで履修して良いです"と言い換えられます。
ただし、必修授業に入っているものは、科目番号が条件を満たしていても選択必修にはカウントされないので気をつけてください。
なお、オーバーする分には問題ないです。ただ、"たとえオーバーして50単位とっても、オーバーした分は卒業単位数にはカウントせず、47までしかカウントしません"という意味です。

では左から3つ目の青枠について。
一番上の段に、総合科目（学士基盤科目）と書かれています。これは、開設授業科目一覧の (1) 総合科目 という部分に掲載されています。学士基盤科目と定められた授業から、卒業までに1~4個履修すれば良いということです。

その下の段は、体育や芸術、国語、外国語など、必修ではない共通科目です。いわゆる第二外国語などもこの枠です。必修じゃない体育では、キャンプやウィンタースポーツがあったと記憶しています。
しかし、単位数に"0~"と書かれていますね。これらは、選択科目ですが別に1単位も取らなくても構わないということです。

左から4つ目の青枠について。
最初の段はなんか色々書いてますが、言い換えれば"他学群開講の授業"というものです。6~15単位取得すればokです。最低でも6単位は取らなきゃいけません。
こちらも科目一覧を見ながら選んでみてください。ただし、授業によっては他学類生は受けられないものや、専門性が強すぎる物などあるので、シラバスを見て自分も取り組めそうだと思えるものを選びましょう。「○○概論」、「○○序説」、「○○入門」などがおすすめです。

中段はGB、GEとありますが、科目番号の説明は先ほどしたので割愛。

下段の"博物館に関する科目"というのは、学芸員になるための授業です。申し訳ないですが、私の身の回りで履修している人がいないのであまり解説できません。
"自由科目（特設）"というのは、キャリア関係の授業や、様々な実践系授業などです。開設授業科目一覧に掲載されています。

中段、下段は"0~"とある通り、履修しなくても良いです。

4, 履修計画を立てよう

1年あたり履修可能な単位数の上限は45単位です（教職、英語の外部試験利用単位認定を除いた合計）。
しかし、成績が良ければ次年度から55単位まで履修できます。これは属にCAP開放と呼ばれています。条件は、履修要覧のあの科目表が載ってるページの少し前。以下の通り。

毎年40単位以上を取ると良いと言われています。3年終了時に120単位以上取っておき、4年生は研究で忙しいので取る授業は限りなく少なくしようというのがスタンダードです。

ここで一つ、大事な話。それは留年についてです。留年の条件については、先ほど何度も登場した履修要覧5-2の自分の学類の表の注釈にあります。

メ創は基本的に3年生までは自動で進級できます。ただし、3年→4年の時に条件があります。それは卒業単位124単位のうち、100単位以上取得していることです。3年生終了までに100単位未満だと、卒業研究ができず留年確定です。気をつけてください。

また、年間取得単位数が15を切ると除籍されます。1回目は頭を下げて、面談で済むとか済まないとか噂がありますが、絶対に15単位より多く取るようにしましょう。この15単位には英検認定などは含まれないので気を付けて。

5, 履修登録をしよう

さて、4では履修計画と時間割の組み方をお伝えしました。それでは、それをもとにいよいよ履修登録です。
履修登録はTwinsというサイトで行います。ログインしたら、履修タブを開くと、時間割が表示されます。ここで履修登録の状況が分かります。

あとは、自分の入れたい授業のモジュールのタブを開き、時間割の対応部分をクリック、科目番号を入力したら時間割へ登録されるので、それで完了です。
これを、各モジュールが始まる前までに行って下さい（厳密には始まって一週間ほどの間は変更可能ですが、早めにやるに越したことはないです）。



ただし、気をつけなければいけないものがあります。
それは、事前登録科目の存在です。



いわゆる抽選科目です。定員が決まっており、それをオーバーしてしまう場合、抽選で履修可能者が決まります。

こちらは、普通の履修登録とは異なり、抽選という過程を挟むため、履修登録の期限がかなり早く設定されています。そして、履修年次1年の専門基礎科目は、ほとんどがこの事前登録科目なのです。（詳しくは入学時に配られる「履修/事前登録」案内を参照）

どれが事前登録科目（=抽選対象）かというと、学士基盤科目すべてと、専門導入科目です。専門導入科目？初めて登場しましたね。
専門導入科目とは、専門基礎・専門科目の一部が属する特別な分類です。専門基礎・専門科目の中でも、全学向けでより入門者へ向けた授業内容です。

ではどれが専門導入なのか。それは科目一覧に載っています。

再び登場、情報メディア入門です。一番右の備考欄を見ると、"専門導入科目"と記載されているのが分かります。備考にこのように記載がある科目は専門導入科目です。

また、「履修/事前登録」案内には、この専門導入科目だけを抽出したページがあるのでそこを見ると良いでしょう。

そして事前登録科目は、先ほど紹介した履修登録とは異なる手順を取る必要があります。

履修タブを開くと、"事前登録"というボタンがあると思います。これを押し、"事前登録"を押します。

すると、モジュール名の一覧が表示されるので、自分が取りたい授業の開始モジュールを押しましょう。あとは、その授業の曜日と時間を押し、優先順位を入力（1つしかない場合は1位にする）し、確認を押して完了です。

この抽選への登録は、自分の学類の選択科目であっても行わなくてはならず、抽選が終わった後だと履修登録ができないので、絶対に期限までに行ってください。
なお、自分の学類の選択必修である場合は抽選にほぼ落ちないらしいです。でも抽選への登録は必要です。




1年生にはもう1つだけ注意点があります。


それは、1年で必修の授業の一部は、履修登録の期限が事前登録科目と同じだということです。

線形代数Aやプログラミング入門ABなどは、定員の関係で履修登録の期限が早めに設定されており、事前登録の締め切りと同日までに履修登録を行わなければ履修できません。プログラミング入門は履修登録以外に、別途フォームの入力もあります。注意してください！！！！4月の間に必修はすべて履修登録しておくと良いでしょう。


これで履修登録は完了です！お疲れ様でした！
